Java é uma linguagem de programação de alto nível, de uso geral, seguro para memória e orientada a objetos. Pretende-se permitir que os programadores escrevam uma vez, executem em qualquer lugar (WORA),o que significa que o código Java compilado pode ser executado em todas as plataformas que suportam Java sem a necessidade de recompilar. As aplicações Java são normalmente compiladas para bytecode que pode ser executado em qualquer máquina virtual Java (JVM), independentemente da arquitetura de computador subjacente. A sintaxe do Java é semelhante a C e C++, mas tem menos facilidades de baixo nível do que qualquer uma delas. O tempo de execução do Java fornece recursos dinâmicos (como reflexão e modificação de código de tempo de execução) que normalmente não estão disponíveis em linguagens compiladas tradicionais.

Java ganhou popularidade logo após o seu lançamento, e tem sido uma linguagem de programação popular desde então. 19] O Java foi a terceira linguagem de programação mais popular em 2022 de acordo com o GitHub. Embora ainda amplamente popular, tem havido um declínio gradual no uso de Java nos últimos anos com outras linguagens usando JVM ganhando popularidade.

Java foi projetado por James Gosling na Sun Microsystems. Foi lançado em maio de 1995 como um componente central da plataforma Java da Sun. Os compiladores Java de implementação original e de referência, máquinas virtuais e bibliotecas de classe foram lançados pela Sun sob licenças proprietárias. Em maio de 2007, em conformidade com as especificações do Processo da Comunidade Java, a Sun havia relicenciado a maioria de suas tecnologias Java sob a licença somente GPL-2.0. A Oracle, que comprou a Sun em 2010, oferece sua própria máquina virtual HotSpot Java. No entanto, a implementação de referência oficial é o OpenJDK JVM, que é um software de código aberto usado pela maioria dos desenvolvedores e é o JVM padrão para quase todas as distribuições Linux.

Java 26 é a versão atual a partir de março de 2026 . Java 8, 11, 17, 21 e 25 são versões de suporte de longo prazo ainda em manutenção.
